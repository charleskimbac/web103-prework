# WEB103 Prework - *Creator Board*

Submitted by: **Charles Kim**

About this web app: **This app allows users to view and manage a list of content creators, including their names, URLs, descriptions, and images. Users can add, edit, and delete creators from the list.**

Time spent: **3** hours
